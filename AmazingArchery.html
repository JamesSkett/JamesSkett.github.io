Amazing Archery

Introduction

Amazing Archery was made for one of my university assignments in my second year. The aim was to create an Android game using Unity and C# that will run on the Nexus 5, the game needed to have tilt controls for aiming and simple gesture controls to simulate archery as best we could.

The game features tilt aiming, dragging down on the screen to set the force that the arrow will be shot at, two different ranges to choose from one more difficult than the other, the harder range includes a moving target as well as a stationary one and lastly a high score board.

GravSwitch - PS4 Game

Introduction

This is a game that I developed for the PS4 as part of a university project. In the game the player can switch the direction that the gravity is pulling in order to make progress in the level.

The game includes basic features like image rendering, platform collisions, sound and animation. The game was developed using C++ along with custom made scripts for loading in different game objects.

Features

Gravity, AABB collisions, sound, spritesheet animation, controller input, particles, load from file scripts for levels and game objects and multithreading for sounds, music and level loading
